# Student Performance Prediction System using Machine Learning

## 📌 Project Overview

This project predicts **student academic performance** using Machine Learning techniques. It includes:

* **Classification:** Predict whether a student will pass or fail.
* **Regression:** Predict the final exam score.

The project demonstrates a complete end-to-end ML workflow including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and insights generation.

---

## 📂 Dataset Information

The dataset contains student-related attributes such as:

* student_id
* gender
* age
* study_hours_per_week
* attendance_rate
* parent_education
* internet_access
* extracurricular
* previous_score
* final_score
* passed

## 🤖 Machine Learning Models Used

### Classification (Pass / Fail Prediction)

* Logistic Regression
* Random Forest Classifier

### Regression (Final Score Prediction)

* Linear Regression
* Random Forest Regressor

---

## 📈 Results

### Classification Model

* Accuracy: **85%**
* Strong pass/fail prediction performance

### Regression Model

* R² Score: **0.756**
* Good final score prediction accuracy

---

## 📊 Key Insights

* Students with higher **previous scores** tend to perform better.
* Higher **attendance rate** improves final results.
* More **study hours per week** generally increase scores.
* Internet access and extracurricular participation may influence performance.
